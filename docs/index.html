O que é o VerificaIdade?

O VerificaIdade é a solução nacional de verificação de idade (18+) baseada em credenciais verificáveis. O governo emite a credencial digital para o cidadão; o seu sistema verifica usando o INJI Verify.

Como funciona

  1. Você integra o INJI Verify SDK (pacote NPM React) na sua aplicação web
  2. O SDK se comunica com o INJI Verify Service (backend Java/Spring Boot que roda na sua VM)
  3. O cidadão apresenta sua credencial via QR Code (desktop) ou Deep Link (mobile)
  4. O Service valida a credencial criptograficamente e retorna o resultado ao SDK

Não é necessário criar um backend customizado para orquestrar a verificação.

Para quem é este portal?

Para desenvolvedores que precisam integrar verificação de idade em aplicações React (Web).
